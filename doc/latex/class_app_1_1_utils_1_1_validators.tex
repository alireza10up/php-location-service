Validates a location search request data.

This method performs validation on a provided location search request data array. It checks for the following\+:


\begin{DoxyItemize}
\item {\ttfamily term}\+:
\begin{DoxyItemize}
\item Required (Not\+Blank)
\item Minimum length of 3 characters (Length)
\item Only contains letters, numbers, spaces, Arabic characters, and percent symbol (Regex)
\end{DoxyItemize}
\item {\ttfamily lat}\+:
\begin{DoxyItemize}
\item Optional
\item Maximum length of 15 characters (Length)
\item If present, must be a valid floating-\/point number (Regex)
\end{DoxyItemize}
\item {\ttfamily lng}\+:
\begin{DoxyItemize}
\item Optional
\begin{DoxyItemize}
\item Maximum length of 15 characters (Length)
\end{DoxyItemize}
\item If present, must be a valid floating-\/point number (Regex)
\end{DoxyItemize}
\end{DoxyItemize}

The method returns an associative array containing validation errors for each invalid field.
