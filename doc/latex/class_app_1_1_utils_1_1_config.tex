Retrieves a configuration value from a file.

This method first calls {\ttfamily get\+File\+Content} to read the contents of the specified configuration file. It then\+:


\begin{DoxyItemize}
\item If {\ttfamily \$key} is null, returns the entire configuration as an associative array.
\item If {\ttfamily \$key} is provided, it retrieves the value associated with that key from the configuration array. If the key does not exist, it returns {\ttfamily null}.
\end{DoxyItemize}
